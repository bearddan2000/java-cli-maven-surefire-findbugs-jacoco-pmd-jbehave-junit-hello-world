# java-cli-maven-surefire-findbugs-jacoco-pmd-jbehave-junit-hello-world

## Description
A POC for maven app using JUnit5
and jbehave framework with findbugs,
jacoco, pmd, and surefire plugin.

## Tech stack
- java
- maven
  - junit
  - surefire
  - findbugs
  - jacoco
  - pmd
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
